# Dynamic Programming 动态规划

---

## ⚡ 速查卡

**识别信号**
- 求最值（最长、最大、最小、最少次数）
- 子问题之间有重叠（暴力递归会重复计算）
- 「是否可行」「有多少种方案」

**解题四步**

```
1. 定义 dp[i] 的含义
2. 找状态转移方程（当前状态怎么从之前状态推来）
3. 确定初始值（base case）
4. 确定遍历顺序
```

**常见子类型**
- **线性 DP**：dp[i] 只依赖 dp[i-1] 或前几个（爬楼梯、最长递增子序列）
- **背包 DP**：选或不选，容量限制
- **区间 DP**：dp[i][j] 表示区间 [i, j] 的最优解
- **路径 DP**：二维网格上的最短 / 最大路径
