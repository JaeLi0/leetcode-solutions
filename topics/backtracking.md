# Backtracking 回溯

---

## ⚡ 速查卡

**识别信号**
- 求所有方案 / 所有组合 / 所有排列
- 需要「撤销」上一步操作
- 决策树：每个节点有若干选择，走到底记录一个结果

**核心模板**

```python
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.append(路径[:])    # 注意要拷贝
        return

    for 选择 in 选择列表:
        # 做选择
        路径.append(选择)
        backtrack(路径, 剩余选择列表)
        # 撤销选择
        路径.pop()
```

**剪枝**：在 for 循环里加条件提前跳过不可能的分支，是回溯的关键优化。

**组合 vs 排列**：组合问题用 `start` 参数控制不往回选，排列问题用 `used` 数组记录用过的元素。
