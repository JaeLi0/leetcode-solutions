# Sliding Window 滑动窗口

---

## ⚡ 速查卡

**识别信号**
- 连续子数组 / 子串 + 满足某个条件的最长 / 最短
- 固定窗口大小，在数组上滑动求最值
- 字符串 anagram / 覆盖子串问题

**核心模板**

```python
# 可变窗口（最常用）
left = 0
for right in range(len(s)):
    # 1. 扩张：将 s[right] 加入窗口
    window[s[right]] += 1

    # 2. 收缩：窗口不合法时，从左缩小
    while 窗口不合法:
        window[s[left]] -= 1
        left += 1

    # 3. 更新答案（此时窗口合法）
    res = max(res, right - left + 1)
```

**复杂度**：时间 O(n)，空间 O(字符集大小)

**和双指针的区别**：滑动窗口维护的是一段连续区间的状态，双指针关注的是两端点的关系。
