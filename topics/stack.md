## ⚡ 速查卡

**识别信号**
- 括号匹配 / 嵌套结构
- 「下一个更大元素」「左边第一个比我大的」
- 需要在处理当前元素时，快速回头看「上一个未处理的」
- 函数调用 / 路径回溯（用栈模拟递归）

**单调栈模板**（处理「下一个更大元素」类问题）

```python
stack = []    # 存下标，单调递减
res = [-1] * len(nums)

for i in range(len(nums)):
    while stack and nums[i] > nums[stack[-1]]:
        idx = stack.pop()
        res[idx] = nums[i]    # nums[i] 就是 idx 的「下一个更大元素」
    stack.append(i)
```

**Python 用 list 模拟栈**：`stack.append()` 入栈，`stack.pop()` 出栈，`stack[-1]` 看栈顶。
